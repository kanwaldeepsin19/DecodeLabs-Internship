# Responsive Portfolio Website

## Description

This project is a responsive portfolio website developed during the DecodeLabs Frontend Development Internship. The website showcases personal information, services, social links, and project details using semantic HTML and CSS. It is designed to adapt to different screen sizes using CSS Grid, Flexbox, and Media Queries, ensuring a smooth user experience across desktop, tablet, and mobile devices.

## Technologies Used

* HTML5
* CSS3
* CSS Grid
* Flexbox
* Media Queries

## Features

* Responsive Navigation Bar
* About Me Section
* Services Section
* Social Links Section
* Responsive Layout
* Semantic HTML Structure

## How to Run

1. Download or clone the repository:

   ```bash
   git clone <repository-url>
   ```

2. Open the project folder.

3. Open `index.html` in any modern web browser.

Alternatively, use VS Code with the Live Server extension:

```bash
Right Click index.html → Open with Live Server
```
